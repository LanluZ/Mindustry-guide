# Print Flush

---

**介绍**

    打印刷新语句  将之前打印的语句刷新在指定message上
        方块:
            message方块

**实例**

游戏内部
```
    Print Flush to message
```
Mindustry (完整)
```
    print val
    printflush message
```
java (完整)
```java
    system.out.print(val);
```

p.s.此语句要与print配合使用
